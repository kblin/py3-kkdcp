Kerberos Key Distribution Center Proxy
======================================

An `aio`-based Python 3 implementation of [MS-KKDCP](http://msdn.microsoft.com/en-us/library/hh553774.aspx),
a protocol to proxy Kerberos ticket requests via HTTP(S).
